发表于 2020-01-25 |

今天在搭设三层的时候遇到个问题，无论是输入WWW,Www,www搜索都是一个结果不区分大小写就导致用户无论密码还是账号大小写都不区分

mysql的模糊查询不区分大小写

public int Login(string username, string userpsd)//登录查询
   {
      string sql = "select count(1) from user where userloginid =@name and userloginpsd = @psd ";
      MySqlParameter spname = new MySqlParameter("@name", MySqlDbType.VarChar) { Value = username };
      MySqlParameter sppsd = new MySqlParameter("@psd", MySqlDbType.VarChar) { Value = userpsd };
      //MySqlParameter sprole = new MySqlParameter("@role", MySqlDbType.VarChar) { Value = name };

      return Convert.ToInt32(DBHelper.ExecuteScalar(sql, spname, sppsd));
  }

解决方法在字段前面都加上binary或者括起来

-- 在每一个条件前加上binary关键字
select * from user where binary username = 'admin' and binary password = 'admin';
-- 将参数以binary('')包围
select * from user where username like binary('admin') and password like binary('admin');

或者

创建表时，直接设置表的collate属性为utf8_general_cs或者utf8_bin；如果已经创建表，则直接修改字段的Collation属性为utf8_general_cs或者utf8_bin。
