增量运算符++

  1. 一元增量运算符++按1递增其操作数，操作数必须是变量，属性访问或者索引器访问

  2. 增量运算符以两种形式进行支持:后缀增量运算符x++和前缀增量运算符++x

    • x++的结果是此操作前的x的值，如以下示例所示:

  5. -int i=3;

  6. -console.writeline(i);—–输出3

  7. -console.writeline(i++);—–输出3

  8. -console.writeline(i);—–输出4

  9. ++x的结果是此操作后的x的值,如以下示例所示:

  10. double a=1.5;

  11. console.writeline(a);—–1.5

  12. console.writeline(++a);—2.5

  13. console.writeline(a);—–2.5

减量运算符–

  • 减和加一样
  1. int i = 3;
  2. Console.WriteLine(i); // output: 3
  3. Console.WriteLine(i–); // output: 3
  4. Console.WriteLine(i); // output: 2
  6. double a = 1.5;
  7. Console.WriteLine(a); // output: 1.5
  8. Console.WriteLine(–a); // output: 0.5
  9. Console.WriteLine(a); // output: 0.5

除法运算符/

  • 对于整数类型的操作数，/ 运算符的结果为整数类型，并且等于两个操作数之商向零舍入后的结果：
  • Console.WriteLine(13 / 5); // output: 2
  • Console.WriteLine(-13 / 5); // output: -2
  • Console.WriteLine(13 / -5); // output: -2
  • Console.WriteLine(-13 / -5); // output: 2
  1. -
    • 若要获取浮点数形式的两个操作数之商，请使用 float、double 或 decimal 类型：
    • Console.WriteLine(13 / 5.0); // output: 2.6
    • int a = 13;
    • int b = 5;
    • Console.WriteLine((double)a / b); // output: 2.6
  8. 对于 float、double 和 decimal 类型，/ 运算符的结果为两个操作数之商：
  9. Console.WriteLine(16.8f / 4.1f); // output: 4.097561
  10. Console.WriteLine(16.8d / 4.1d); // output: 4.09756097560976
  11. Console.WriteLine(16.8m / 4.1m); // output: 4.0975609756097560975609756098
  12. 如果一个操作数为decimal另一个数不能为float和double 这2个不能隐式转化为decimal 必须将float
  13. 和double显示转换为decimal

余数运算符%

  • 对于整数类型的操作数，a % b 的结果是 a (a / b) * b 得出的值。 非零余数的符号与左侧操作数的符号相同，如下例所示：
  • Console.WriteLine(5 % 4); // output: 1
  • Console.WriteLine(5 % -4); // output: 1
  • Console.WriteLine(-5 % 4); // output: -1
  • Console.WriteLine(-5 % -4); // output: -1
