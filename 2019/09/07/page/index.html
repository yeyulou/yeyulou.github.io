C sharp 布尔逻辑运算符和位移，按位求补

发表于 2019-09-07 
int x = 22;
int y = 2;
if (x >= 33 && ++y > 0)
{
    Console.WriteLine(y);
}
Console.WriteLine(y);
ouput：2

if (x >= 33 & y++ > 0)
{
    Console.WriteLine(y);
}
Console.WriteLine(y);

Console.ReadKey();
output：3

&和&&区别，&具有短路功能当第一个条件为假时第二个条件就不判断而&继续执行y++,

-布尔逻辑按位逻辑and &就是按位与，需要判断2个值
-条件与 &&就是并且，表示同时满足&&前后的表达式或者值为真才执行
|和&优先级高于&&和||，但&&效率相对于&好点

p    q    p & q    p | q    p ^ q
0    0      0          0          0
0    1      0          1          1
1    1      1          1       0
1    0      0          1       1

假设如果 A = 60，且 B = 13，现在以二进制格式表示，它们如下所示：

A = 0011 1100

B = 0000 1101

A&B = 0000 1100

A|B = 0011 1101

A^B = 0011 0001

~A = 1100 0011

 运算符                                                             
 &    如果同时存在于两个操作数中，二进制 AND 运算符复制一位到结果中。    
 |    如果存在于任一操作数中，二进制 OR 运算符复制一位到结果中。
 ^    如果存在于其中一个操作数中但不同时存在于两个操作数中，二进制异或运算符复制一位到结果中。    
 ~    按位取反运算符是一元运算符，具有"翻转"位效果，即0变成1，1变成0，包括符号位。    (~A ) 将得到 -61，即为 1100 0011，一个有符号二进制数的补码形式。
<<    二进制左移运算符。左操作数的值向左移动右操作数指定的位数。    
>>    二进制右移运算符。左操作数的值向右移动右操作数指定的位数。

    int a = 60;            /* 60 = 0011 1100 */  
    int b = 13;            /* 13 = 0000 1101 */
    int c = 0;           

     c = a & b;           /* 12 = 0000 1100 */ 
     Console.WriteLine("Line 1 - c 的值是 {0}", c );

     c = a | b;           /* 61 = 0011 1101 */
     Console.WriteLine("Line 2 - c 的值是 {0}", c);

     c = a ^ b;           /* 49 = 0011 0001 */
     Console.WriteLine("Line 3 - c 的值是 {0}", c);

     c = ~a;               /*-61 = 1100 0011 */
     Console.WriteLine("Line 4 - c 的值是 {0}", c);

     c = a << 2;     /* 240 = 1111 0000 */
     Console.WriteLine("Line 5 - c 的值是 {0}", c);

     c = a >> 2;     /* 15 = 0000 1111 */
     Console.WriteLine("Line 6 - c 的值是 {0}", c);
     Console.ReadLine();

^ 逻辑XOR 在bool判断中和(不等运算符)!=相同 如果为true且false 或者 false且true为真（true）

Console.WriteLine(true ^ true);    // output: False
Console.WriteLine(true ^ false);   // output: True
Console.WriteLine(false ^ true);   // output: True
Console.WriteLine(false ^ false);  // output: False

bool？就是bool能为null 注意!!!!!!条件逻辑运算符 && 和 || 不支持 bool? 操作数。

bool? test = null;
Display(!test);         // output: null
Display(test ^ false);  // output: null
Display(test ^ null);   // output: null
Display(true ^ null);   // output: null 

void Display(bool? b) => Console.WriteLine(b is null ? "null" : b.Value.ToString());

按位求补运算符 ~（也可用于析构函数具体请看析构函数blog）取相反 0变成1 1变成0也包含符号位（+ -）

uint a = 0b_0000_1111_0000_1111_0000_1111_0000_1100;
uint b = ~a;
Console.WriteLine(Convert.ToString(b, toBase: 2));
// Output:
// 11110000111100001110000111110011
