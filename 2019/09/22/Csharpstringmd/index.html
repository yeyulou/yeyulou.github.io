结构体是值类型数据结构而类是引用类型且结构不支持继承和声明默认的构造函数
结构可以有方法，字段，索引，属性，预算符方法和事件
不能做为其他结构和类的基础结构
可以实现一个或多个接口但其成员不能abstract，virtual和protected
可以不用new实例化但假如不实例化只有所有字段都被初始化后字段才能被赋值和使用

结构体之前说到了不用new赋值
结构体在声明的字段的时候不能赋予初值但是类是可以的

struct test001
{
private int aa = 1;
}
编译器会报错
class test002
{
private int aa = 1;
}
  1. 且结构体的构造函数中必须为所有字段赋值，类不用
  2. 类的对象存储在堆空间中，
  3. 结构体存在在栈空间中，堆空间大但访问速度慢，栈相反空间小速度快
  4. int double bool都是结构类型 类在堆空间分配空间，栈只是保存引用

当堆栈空间有限时用类好点，矩形点和颜色声明一个含有许多颜色对象的数组，CLR(运行环境)需要为每个对象分配内存则结构更好
